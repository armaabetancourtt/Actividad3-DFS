# Actividad 4: Desarrollo Full Stack

# Gestor de Tareas y Productos - Express

Este proyecto es una aplicación backend desarrollada con Node.js y Express que se ejecuta en el despliegue en la nube de Render. Permite autenticación de usuarios, gestión de tareas y productos.

---

## Requisitos

Antes de ejecutar el proyecto, asegúrese de tener instalado:

- Node.js
- npm
---

## Ejecución del proyecto en Cloud

### 1. Abrir el siguiente enlace: https://actividad4-dfs.onrender.com/
 
Espere de 30 a 60 segundos y la página automáticamente se mostrará
